VC Radar · 使用流程

4 步上手，
开始高效找工

从安装到每天用起来，只需要 5 分钟

1
安装 Chrome 插件
下载仓库后，在 Chrome 开发者模式下加载 extension/ 文件夹，30 秒完成安装
Chrome 浏览器 开发者模式
2
填写 API Keys
点击浏览器右上角图标，填入 Anthropic API Key 和 Notion Integration Token，保存即可
Anthropic API Notion Token
3
浏览职位，AI 自动分析
打开 LinkedIn 或 Boss直聘，公司名旁自动出现融资角标，悬停陌生公司触发 Claude AI 深度研究
LinkedIn Boss 直聘 AI 评分
4
一键记录到 Notion
看到心仪职位，点击 📌 按钮，公司名、职位、来源、日期自动写入你的 Notion 求职数据库
自动同步 求职追踪
